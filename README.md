# Amazon Prime Movies & TV Shows Data Analysis

## Project Overview
This project performs Exploratory Data Analysis (EDA) on the Amazon Prime Movies and TV Shows dataset to understand content trends, genre distribution, and platform growth.

## Dataset
The dataset contains information about movies and TV shows available on Amazon Prime, including:
- Title
- Genre
- Release Year
- Cast and Credits
- Content Type

## Tools Used
Python  
Pandas  
Matplotlib  
Seaborn  
Google Colab

## Key Analysis Performed
- Data Cleaning and Preprocessing
- Content Type Distribution (Movies vs TV Shows)
- Release Year Trends
- Genre Analysis
- Cast and Credits Analysis

## Project Files
- `amazon_prime_movies_tv_shows_eda.ipynb` → Complete EDA analysis notebook
- `cleaned_titles.csv` → Cleaned titles dataset
- `cleaned_credits.csv` → Cleaned credits dataset
